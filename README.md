# Tales of Symphonia — Run Tracker

Tracker de run **mono-fichier** pour suivre une partie de *Tales of Symphonia* : un seul `index.html`, **zéro dépendance, zéro build**. Ouvre-le dans un navigateur, c'est tout.

La progression est sauvegardée en `localStorage`, avec **export / import**.

## Onglets

- **Checklist** — progression et boss du run
- **Affection** — suivi du système d'affinité entre personnages
- **Arms** — armes
- **Cuisine** — recettes
- **Référence** — infos utiles en cours de partie
- **Réglages** — sauvegarde, export / import des données

## Utilisation

Ouvre `index.html` dans n'importe quel navigateur moderne (ou via la page déployée). Rien à installer.
